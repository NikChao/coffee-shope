# Barista
Barista is a coffee-shope specific tool for scaffolding and building components.

## Commands

### Creating a component
**Command:** `barista create` or `barista create component`

**Flags**
- `--no-typescript`
- `--no-storybook`
- `--organisation-name`
- `--packages-dir`

### Building a component
**Command:** `barista build`

### Creating a util
**Command:** `barista create util`

**Flags**
- `--no-typescript`
- `--no-storybook`
- `--organisation-name`
- `--packages-dir`

### Building a component
**Command:** `barista build util`

**Flags**
- `--no-typescript`
- `--no-storybook`
